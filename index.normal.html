# Attributes

Attribute Defaults Name Description
data-percent (obligatory) N/A Percent of the circular bar
data-stroke stroke Border color of the circular bar (default: #2ecc71)
data-stroke-width strokeWidth Border width of the circular bar (default: 10)
data-text text Text of the inner label (default: none)
data-text-color textColor Color of the label text (default: #2ecc71)
data-dimensions dimensions Dimensions of the circular progress bar.(default: 200px)
data-show-path showPath Show progress path true/false (default: true)
data-path-stroke pathStroke CSS color for the path stroke (default: rgb(0,0,0,0.1))
data-text-font textFont Font of the inner label (default: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif)
data-text-size textSize Size of the inner label (you can use every units of css) (default: 20px)
data-stroke-linecap strokeLinecap Stroke linecap (round,square,butt,...) (default: round)
N/A calcDashoffset Function to calculate the stroke-dashoffset of the circle. Arguments passed: pathLenght

To set your defaults, you have to set the circularProgressDefaults variable before including the script into the document using the upper names. Example: 
var circularProgressDefaults = {
    "stroke":"rgb(255,255,255)", // White stroke
    "strokeWidth":20, // Fatter stroke
    "text":"default text", // Default text
    "textColor":"rgb(255,255,255)", // White text color
    "dimensions": "100px", // Tinier circular progress bar
    "showPath": true, // Show progress path
    "pathStroke": "rgba(255,255,255,0.2)" ,// Lighter path stroke 
    "calcDashoffset": function(pathLenght) {return pathLenght / 6;} // Slightly inclined circular bar progress value (DANGEROUS: edit this only if you know what you're doing)
};

document.getElementById('testProgress').setPercent(100);

# Getting started

To use the code, you have to include it in your HTML document.
Put the script tag at the end of the body tag:
<script defer src="https://github.pato05mc.tk/circular-progress.js/src/circular-progress.min.js"></script>


Alternatively, you can download it from the repository.

To put a new circular progress, all you have to do is put a div with a circularProgress class and a data-percent attribute 
<div class="circularProgress" data-percent="[number between 1 and 100]"></div>



To create new circularProgress elements in Javascript, you can run:
circularProgress.new(divElement, progress, label);


Then you have to append the returned element to the body. In this case, you don't have to set the circularProgress class. 
document.getElementById('newProgTest').appendChild(circularProgress.new(document.createElement('div'), 90, 'New Progress Bar'));
Run Reset
To set other options in the 
circularProgress.new()
function, you can set the attributes in the new created div element before calling the function
