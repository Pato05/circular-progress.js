# Attributes

AttributeDescription
data-percent (obligatory)Percent of the circular bar
data-strokeBorder color of the circular bar (default: #2ecc71)
data-stroke-widthBorder width of the circular bar (default: 10)
data-textText of the inner label (default: none)
data-text-colorColor of the label text (default: #2ecc71)
data-dimensionsDimensions of the circular progress bar.(default: 200px)
data-show-pathShow progress path true/false (default: true)
data-path-strokeCSS color for the path stroke (default: rgb(0,0,0,0.1))

You can always set your default by setting the circularProgressDefaults variable before including the file into your document. Example:
var circularProgressDefaults = {
    "stroke":"rgb(255,255,255)", // White stroke
    "strokeWidth":20, // Fatter stroke
    "text":"default text", // Default text
    "textColor":"rgb(255,255,255)", // White text color
    "dimensions": "100px", // Tinier circular progress bar
    "showPath": true, // Show progress path
    "pathStroke": "rgba(255,255,255,0.2)" // Lighter path stroke
}

# Functions


element.setPercent(value);


This function will set the circular percent to a value.
document.getElementById('testProgress').setPercent(100);

element.setLabel(newLabel);


This function will set the new label text, and create a label if there isn't one.
document.getElementById('testProgress').setLabel('New Label');

# Getting started

To use the code, you have to include it in your HTML document.
<link rel="stylesheet" href="https://github.pato05mc.tk/circular-progress.js/src/circular-progress.min.css">
<script defer src="https://github.pato05mc.tk/circular-progress.js/src/circular-progress.min.js"></script>


Alternatively, you can download it from the repository.

To put a new circular progress, all you have to do is put a div with a circularProgress class and a data-percent attribute
<div class="circularProgress" data-percent="[number between 1 and 100]"></div>



To create new circularProgress elements, you can run:
circularProgress.new(divElement, progress, label);



Then you have to append the returned element to the body. In this case, you don't have to set the circularProgress class.
document.getElementById('newProgTest').appendChild(circularProgress.new(document.createElement('div'), 90, 'New Progress Bar'));
